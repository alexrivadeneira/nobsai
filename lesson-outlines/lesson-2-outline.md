# Lesson 2: Your First Real Automation — Outline

Working title: "Put the Function to Work"
Path step 2 · promise: "stop chatting with AI and start making it do work for you — something real running on your computer in 20 minutes"

## 1. Hook

You've been using AI in manual mode: you type, it answers, you type again. Today you stop being the typist.

## 2. The big idea: a prompt is a program

Callback to Lesson 1's Round Trip widget: remember the green text the app stitched around your words? Somebody *wrote* that. Instructions written once, applied to every message — that's all a "system prompt" is, and it's the whole trick behind every AI product. Today you write your own.

## 3. Experiment 1: "Be the Wrapper" widget ✅ built

`be-the-wrapper-widget.html` — pick a role (home cook / poet / nutrition coach) and a format (one sentence / numbered list), watch them assemble into the green system prompt, run the same fixed message, compare outputs side by side. Payoff: the message never changed, the function never changed — only the job description did.

## 4. Build it for real (the 20-minute deliverable)

- Warm-up: a reusable prompt saved in a note. Works on any chatbot, zero paywall, hammers home "a prompt is a program — you just saved a program in a text file."
- Main build: a **Gemini Gem** — free on every Google account, unlimited, web-only creation. Name + instructions + test panel mirrors the widget layout. Write a job description for a real chore (meeting-notes formatter, email tone fixer, weekly meal planner); stress-test with 3 real inputs.
- Sidebar: Claude users → Projects (free with limits). ChatGPT users → custom GPTs require Plus ($20/mo), so not the default path; free-tier custom instructions apply globally (wrong shape for a specialist).

## 5. When does it count as automation?

Honest distinction: a saved specialist still needs you to push the button. Real automation = the *computer* sends the message. Show one scheduled task (e.g. a daily digest) as the first taste. Sets up Lesson 3 — "Meet your first agent": letting the function push buttons itself.

## 6. Catch it in the wild

Gmail's draft replies, Notion AI, customer-service bots — every "AI feature" is someone's job description stapled to the same frozen function. Once you see wrappers, you can't unsee them.

## 7. Decode the buzzwords

Prompt engineering · system prompt · custom GPT · agent · "powered by AI"
